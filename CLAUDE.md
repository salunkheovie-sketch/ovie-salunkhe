# Ovie Salunkhe Cricket Portfolio - AI Assistant Guide

This file provides context and instructions for AI coding assistants working on this repository.

## Project Overview
This project is a Next.js-based portfolio for **Ovie Salunkhe**, a rising Under-17 cricket player from Mumbai. It tracks his career milestones, match statistics, and training journey toward the highest levels of Indian cricket.

## Development Commands
- `npm run dev`: Start the development server on localhost:3000.
- `npm run build`: Build for production.
- `npm run lint`: Run ESLint.

## Code Style & Conventions
- Use **TypeScript** strictly. Avoid using `any`.
- Use **Next.js App Router** conventions (e.g., `app/page.tsx`, `app/layout.tsx`).
- **Styling:** Focus on rich aesthetics and modern design. The UI should feel premium, dynamic, and engaging. Utilize micro-animations, high-quality typography, and responsive layouts.
- **Component Architecture:** Keep React Server Components (RSC) as the default. Use the `'use client'` directive only when client-side interactivity, state, or hooks are required.
