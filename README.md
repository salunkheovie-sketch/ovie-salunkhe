# Ovie Salunkhe - Cricket Portfolio

Welcome to the official portfolio and career tracking platform for Ovie Salunkhe.

## About Ovie Salunkhe

Ovie Salunkhe is a rising cricket talent from Mumbai, currently representing the city in the Under-17 circuit, including the prestigious MCA West Zone Championship squad. Known for a fierce passion for the game and sharp on-field instincts, Ovie balances rigorous academy training with high-stakes competitive matches.

## Platform Purpose

This platform is dedicated to tracking Ovie's ongoing journey toward the highest levels of Indian cricket. Here you will find:
- **Career Milestones:** Key achievements, awards, and significant moments in Ovie's cricketing journey.
- **Match Statistics:** Detailed records and statistics from recent and historical matches.
- **Ongoing Journey:** Updates on training, upcoming matches, and progress.

## Tech Stack

This project is built using:
- [Next.js](https://nextjs.org/)

## Getting Started

First, install dependencies (if you haven't already):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
